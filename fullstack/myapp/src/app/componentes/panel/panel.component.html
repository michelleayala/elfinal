Bienvenido administrador

Utiliza este icono para desplegar o contraer la barra de navegación, esto te ayudará a tener una vista completa con toda las pestañas disponibles o te dará una vista más limpia de tu pantalla si así lo deseas.


Utiliza este icono para desplegar la pestaña de Productos, dentro de esta pestaña podrás encontrar todo lo relacionado con la administración de tus productos.


Utiliza este icono para desplegar la pestaña de Categorias, dentro de esta pestaña podrás encontrar los productos que tienes en cada categoría.


Utiliza este icono para desplegar la pestaña de usuarios, dentro de esta pestaña podrás ver y crear usuarios los cuales tendrán acceso a este panel administrativo, ten en cuenta de que ellos podrán editar la información de tus productos, pero no podrán crear nuevos usuarios.


Utiliza este icono para desplegar la pestaña de Configuracion, dentro de esta pestaña podrás encontrar todo lo relacionado con la configuración de colores e imágenes promocionales de tu tienda.


Utiliza este icono solo si quieres cerrar sesion en el panel administrativo, ya que este té redirecciona a la página de tu tienda.
